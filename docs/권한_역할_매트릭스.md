# 권한/역할 매트릭스 (MVP-1)

## 역할 정의
- FT(컨설턴트): 프로젝트 운영, 데이터 업로드/편집/승인, 리포트 배포
- 리더(팀장): 팀 리포트 + 리더 리포트 열람
- 구성원: 팀 리포트 + 개인 리포트 열람

## 한 줄 요약
- FT: 전체 관리자(생성/편집/승인/배포 + 모든 리포트 열람)
- 리더: 팀 요약 + 리더 리포트 열람(개인 리포트는 제외)
- 구성원: 팀 요약 + 본인 개인 리포트만 열람

| 기능/리소스 | FT | 리더 | 구성원 |
| --- | --- | --- | --- |
| 프로젝트 생성/관리 | O | X | X |
| 팀 컨텍스트 입력/수정 | O | X | X |
| 파일 업로드(PDF/XLSX) | O | X | X |
| 리포트 초안 생성 | O | X | X |
| 리포트 편집/승인 | O | X | X |
| 팀 리포트 열람 | O | O | O |
| 개인 리포트 열람 | O | X | 본인만 |
| 리더 리포트 열람 | O | O | X |
| FT-only 리포트 열람 | O | X | X |
| PDF 다운로드 | O | O | O(권한 범위 내) |
| 공유 링크 생성 | O | X | X |
| 공유 링크 열람 | O | O | O(권한 범위 내) |
| 배포 이력 조회 | O | X | X |

## 데이터 보호 원칙
- 개인 상세 정보는 FT 전용
- 리더/구성원은 익명 집계 또는 권한 범위 내 요약만 접근
- 공유 링크는 역할별 리포트 범위를 제한

## 로그/감사
- FT 편집/승인 이력 저장
- 리더/구성원 열람 로그 저장
