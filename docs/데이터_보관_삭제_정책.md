# 데이터 보관/삭제 정책 (MVP-1)

## 결정된 사항
- 보관 기간: 12개월(FT/참여자 접근 가능)
- 보관 기간 만료 후 처리: 아카이브(접근 차단)
- 접근 권한: 아카이브 상태에서 FT/참여자 모두 접근 불가
- 복구: 관리자만 가능(요청/승인 로그 남김)
- 복구 가능 기간: 무기한
- 관리자 정의: FT 중 권한이 부여된 관리 FT
- 삭제 권한: 관리자만 삭제 가능
- 삭제 방식: 7일 유예 후 삭제
- 백업 보관 기간: 30일
- 로그 보관 기간: 24개월
- 익명화/비식별화: 가명처리(관리자만 복구 가능)
- 재진단/팀 이동 시 정책: 이전 자료 복구 후 활성 접근 기간 12개월 자동 연장
- 복구 SLA(운영 기준): 요청 후 24~48시간 내 복구

## 참고
- 아카이브 상태의 데이터는 일반 사용자에게 노출되지 않으며, 복구 시 기록을 남김
