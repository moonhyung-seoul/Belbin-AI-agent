# 디자인 브리프: 리포트 템플릿 (신지영님 전달용)

목적: Belbin AI Agent 리포트(팀/개인/리더/FT) 출력용 **디자인 템플릿**을 만들기 위한 기준/제약/필수 요소를 정리.

---

## 1) 리포트 종류(4종)
1. 팀 리포트(Team pack)
2. 개인 리포트(팀원용)
3. 리더 리포트(리더용)
4. FT-only 리포트

---

## 2) 공통 디자인 제약
- **톤**: 중립적, 비판 없는 컨설팅 톤
- **언어**: 한국어 기본
- **출력 목표**: PDF 1~2p (각 리포트)
- **중요**: 핵심 메시지 → 근거 → 액션 순으로 읽히도록 구성
- **표시 규칙**: FT 전용 요소는 별도 시각 구분(예: 라벨/배지)
- **근거 표기**: 주요 주장 옆에 [SRC: …] 표기 가능하도록 공간 확보

---

## 3) 팀 리포트(Team pack) 구성
- 팀 성향 한 문장
- 팀 강점/리스크 Top 2~3
- 팀 역할 밸런스(부재/과잉)
- 개인-팀 비교(표시/강조 기준)
- 팀 강점 발휘 순간 / 팀 잠재 이슈 순간(4~5)
- 팀 공동 액션 안건(3개)

**레이아웃 제안**
- 상단: 팀 성향 한 문장
- 중단 좌: 강점/리스크, 중단 우: 역할 밸런스
- 하단: 개인-팀 비교 + 시나리오 + 액션 안건

---

## 4) 개인 리포트(팀원용) 구성
- Self vs Observer 요약
- Top/Bottom 역할
- 특징 키워드
- 기여 포인트(업무 장면)
- 주의/리스크 신호
- 행동 제안(강점 강화/약점 관리 각 2~3)

**레이아웃 제안**
- 상단: Self vs Observer 요약
- 중단: Top/Bottom + 키워드
- 하단: 행동 제안 카드형 2블록

---

## 5) 리더 리포트(리더용) 구성
- Self vs Observer 요약
- 현재 기여 포인트
- 리스크/블라인드스팟
- 시스템으로 보완하는 리더 행동 가이드
- (옵션) 코칭 사전자료

---

## 6) FT-only 리포트 구성
- 퍼실리테이션 주의사항
- 참석자별 특이사항 1-pager

---

## 7) 데이터 표시 규칙
- 개인-팀 비교: **35% 이상 표시**, **200% 이상 강조**
- 팀 평균 0인 역할은 **부재 역할 / 희소 역할 보유**로 표시
- 액션 안건: **왜/무엇/어떻게(옵션 2~3)/필요 행동·팀역할**

---

## 8) 산출물 형식
- 디자인 템플릿: Figma/PowerPoint/HTML 중 선택
- 최종 출력: PDF
